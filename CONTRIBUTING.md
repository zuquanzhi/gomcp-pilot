# 贡献指南 (Contributing to gomcp-pilot)

感谢您对 gomcp-pilot 的关注！我们需要您的帮助来让这个项目变得更好。

## 如何贡献

### 1. 提交 Issue

如果您发现了 bug 或有新的功能建议，请先搜索现有的 [Issues](https://github.com/zuquanzhi/gomcp-pilot/issues) 看看是否已经有人提出。如果没有，请创建一个新的 Issue，并详细描述您的问题或建议。

### 2. 提交 Pull Request (PR)

如果您想直接贡献代码：

1.  Fork 本仓库。
2.  创建一个新的分支 (e.g. `git checkout -b feature/amazing-feature`).
3.  提交您的更改 (e.g. `git commit -m 'Add some amazing feature'`).
4.  推送到您的分支 (e.g. `git push origin feature/amazing-feature`).
5.  提交一个 Pull Request。

## 开发流程

请参考 [README.md](README.md) 中的 "快速开始" 部分来搭建本地开发环境。

### 代码风格

*   Go 代码请遵循 standard Go formatting (`go fmt`).
*   提交信息请清晰明了。

## 行为准则

请保持友善和尊重。我们希望建立一个包容的社区。

---

再次感谢您的贡献！
